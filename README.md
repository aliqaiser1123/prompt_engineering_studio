# 🚀 Prompt Engineering Studio

An interactive AI application for designing, testing, and optimizing prompts using professional prompt engineering techniques. Experiment with AI roles, reusable prompt templates, inference parameters, structured outputs, and conversation memory across multiple LLMs.

## ✨ Features

- 🤖 Multi-LLM Support
- 🎭 AI Role Selection
- 📝 Prompt Templates
- 🌡️ Temperature Control
- 📄 Multiple Output Formats
- 💬 Conversation Memory
- 📥 Export Chat
- 🗑️ Clear Chat History
- ⚡ Fast Inference with Groq API

## 🛠️ Tech Stack

- Python
- Streamlit
- Groq API
- python-dotenv

## 🚀 Run Locally
- Add you GROQ API KEY to .env file
```bash
pip install -r requirements.txt
streamlit run app.py
```
